# Checklist de entrega — Certa Hora

## Antes da apresentação

- [ ] Reiniciar o sistema com `iniciar.bat`.
- [ ] Executar `executar_testes.bat` e confirmar seis testes aprovados.
- [ ] Gerar um backup pelo painel do gestor.
- [ ] Conferir as credenciais de demonstração.
- [ ] Validar login do gestor e do funcionário.
- [ ] Confirmar que a porta 8000 está disponível.
- [ ] Abrir o relatório técnico em Word.

## Fluxos para demonstrar

- [ ] Cadastro e edição de funcionário.
- [ ] Configuração da jornada.
- [ ] Registro de ponto.
- [ ] Confirmação de marcação próxima.
- [ ] Jornada incompleta.
- [ ] Hora extra e justificativa.
- [ ] Análise administrativa sem alteração das horas.
- [ ] Correção com auditoria.
- [ ] Filtros e exportação Excel.
- [ ] Auditoria e backup.

## Limites que devem ser informados

- [ ] Projeto acadêmico, não homologado como REP-P.
- [ ] Execução local em HTTP.
- [ ] SQLite adequado à demonstração e a instalações pequenas.
- [ ] Regras coletivas e particularidades empresariais exigem validação especializada.
